# Mowgli Sample Library

Sample library of the Mowgli synth

Build repo can be found here: https://bitbucket.org/lukeareid/mowgli/src/main/

## Patches

All patches have been documented here, and will be kept up-to-date: https://docs.google.com/spreadsheets/d/1huNAUYcMz1X1Fti60tfZ4A_Ux2M2KIYcLl82OxsYylc/edit?usp=sharing

## Banks

* Bank A - Single wave forms
* Bank B - Dual (or multi) wave forms
* Bank C - Filter character
* Bank D - Envelope character
* Bank E - Unison
